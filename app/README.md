# ContextLink Review App
This folder contains an early Streamlit prototype for human-in-the-loop review of ambiguous entity matches.
The app is designed to help researchers inspect candidate matches generated by the pipeline, confirm or reject them, and export reviewed results.

## Planned Workflow
1. Load candidate matches from the pipeline.
2. Display original names, cleaned names, and suggested matches.
3. Allow human reviewers to confirm, reject, or edit matches.
4. Export reviewed results for downstream data cleaning.

## Status
Early prototype. The current version is being developed with synthetic data.
